# Moat CLI User

Command line interface to Dusk Citadel User

Available commands:

- submit a license request to blockchain
- list user's licenses
- use license
- request service from SP (Off-Chain)

## Submit a license request to blockchain

Arguments:
- address of the license provider

## Use license
